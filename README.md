# জেনারেটর বট - Generator Bot

একটি বহুভাষিক টেলিগ্রাম বট যা QR কোড ও PDF নিয়ে কাজ করে।

## বৈশিষ্ট্য
- 🔳 QR কোড জেনারেটর
- 📄 PDF কনভার্টার (ছবি→PDF, PDF→ছবি)
- 🔗 PDF মার্জ, ✂️ স্প্লিট, 🔐 পাসওয়ার্ড
- 🌐 বাংলা ও ইংরেজি ভাষা সাপোর্ট
- ⚙️ ইউজার ফ্রেন্ডলি মেনু

## ডেপ্লয়মেন্ট
Render-এ ডেপ্লয় করতে Environment Variables সেট করুন:
- `BOT_TOKEN` = আপনার বট টোকেন
